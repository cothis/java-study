리스트 작성시 사용태그(ol, ul)

ol, ul 자녀태그는 li 태그만 사용

순서가 있는 리스트(목록): ol - ordered list

  1. 사과
  2. 바나나
  3. 딸기

순서가 없는 리스트(목록): ul - Unordered list

  • 사과
  • 배
  • 복숭아
  • 수박

계층구조 목록 작성 : ul + ol

  • 사과
    1. 최상품사과
    2. 상품사과
    3. 보통사과
    4. 불량사과
      • 벌레먹은 사과
      • 썩은 사과
      • 덜익은 사과
  • 배
    1. 최상품배
    2. 상품배
    3. 보통배
    4. 불량배
