CSS display: block, inline, inline-block

div 태그는 block 속성

block 속성은 브라우저 좌측부터 우측까지 영역을 차지

block 속성은 크기를 가짐 - width, height 지정

div는 block 속성

span 태그는 inline 속성

inline 속성은 라인 내에서 영역을 차지(데이터 크기만큼)

inline 속성은 내용(contents)의 크기로 정해짐

span은 inline 속성. 그래서 한 줄로 표시된다.

inline-block 속성

inline-block 속성은 라인 내에서 영역을 차지(inline)

inline-block 속성은 크기를 가짐 - width, height 지정(block)

inline-block은 전체 영역을 차지하지 않고 한 줄에 표시되지만,
가로, 세로 크기
를 가진다.
